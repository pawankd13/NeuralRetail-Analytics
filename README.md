#  Neural Retail Analytics Project

## Overview

This project analyzes retail transaction data to generate business insights using data analytics and machine learning techniques.

The project covers:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Customer Segmentation (RFM)
* Sales Forecasting
* Customer Churn Analysis
* Inventory Analysis
* Streamlit Dashboard Development
